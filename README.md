# React Sample App

## Run the application locally

### Dependencies
- Node v10+
- cloned repository

### Running

```bash
# after cloning the repo:

# install packages
npm i

# compile the project
npm run compile

# run local server
# head over to http://0.0.0.0:8020
npm run dev

# run json-server
# head over to http://0.0.0.0:8021
npm run server

```
